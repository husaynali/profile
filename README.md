## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
streamlit run app.py
```

3. Open your browser to `http://localhost:8501`

## Customization

- **Profile Info**: Edit the profile section in `app.py`
- **Projects**: Update the `projects` list with your own work
- **Skills**: Modify the skills tags in the Skills tab
- **Blogs**: Add your blog posts to the `blogs` list
- **Colors**: Adjust the Twitter blue (`#1d9bf0`) in the CSS if needed

## Deploy

Deploy to Streamlit Cloud:
1. Push this repo to GitHub
2. Go to share.streamlit.io
3. Connect your repo
4. Deploy!
